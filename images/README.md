# Images Directory

Place your logo image file here with the filename: **nixi_logo1.jpg**

The logo is used in:
- Home page (index.php)
- Navigation header (all pages using renderHeader function)

**Supported formats:** JPG, JPEG, PNG (recommended: JPG)

**Recommended dimensions:** 
- Width: 200-300px
- Height: 50-100px
- Aspect ratio: Maintain your logo's original proportions

The logo will be automatically resized by CSS to fit the design.
